# 🍔 FoodApp

FoodApp is a modern food ordering web application built using **React.js**. It fetches data from Swiggy’s API to display restaurant details and their menus in an interactive, user-friendly interface.

---

## 🚀 Features

- 🏬 Explore Restaurants
- 📃 View Detailed Menus
- 🍽️ Dish Information with Images and Prices
- ⚡ Shimmer UI while loading
- 🔍 Routing via React Router
- 💻 Responsive Design

---

## 🛠️ Tech Stack

- **React.js**
- **React Router**
- **CSS3**
- **JavaScript (ES6+)**
- **Swiggy Public API**
